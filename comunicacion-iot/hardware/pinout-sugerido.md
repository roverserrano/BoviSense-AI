# Pinout Confirmado

Este proyecto usa numeracion fisica BOARD en Jetson.GPIO.

| SX1278 | Jetson Orin Nano |
| --- | --- |
| MOSI | pin 19 / SPI0_MOSI |
| MISO | pin 21 / SPI0_MISO |
| SCK | pin 23 / SPI0_SCK |
| NSS/CS | pin 24 / SPI0_CS0 |
| GND | pin 25 |
| VCC | pin 17 / 3.3V |
| RST | pin 22 |
| DIO0 | pin 13 |

Validar alimentacion, antena y continuidad antes de encender.
