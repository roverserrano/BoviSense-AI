# Advertencias de Seguridad

- No usar 5V con SX1278.
- No ejecutar scripts con `sudo` desde este modulo.
- No conectar o desconectar con el sistema energizado si no se controla ESD.
- No transmitir sin antena adecuada.
- No modificar Jetson-IO automaticamente.
- Verificar normativa local para 433 MHz.
- Mantener potencia TX moderada durante pruebas.
